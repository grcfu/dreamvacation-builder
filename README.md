# ✈️ The Voyager Edit
> A digital journal for the aesthetic wanderer. Choose where you want to begin.

**The Voyager Edit** is an interactive web experience designed for modern travelers. It blends editorial design with functional utility, allowing users to "check in" to any city, discover AI-curated gems, and document their journey through a digital boutique and sketchbook.

## ✨ Features
* **AI-Powered Itineraries:** Real-time suggestions for trendy, aesthetic activities via OpenAI.
* **Atmospheric UI:** Dynamic environmental effects (rain or sun motes) that trigger based on local weather data.
* **Interactive Journaling:** A custom canvas for "doodling" and packing travel essentials.
* **Vintage Boarding Pass:** Generates a unique, validated ticket with randomized flight data and barcodes.
* **Aurora Glass Reset:** A high-fidelity, iridescent UI reset for planning new voyages.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Scroll-Snap, Flexbox, Custom Properties)
* **Logic:** Vanilla JavaScript (ES6+)
* **APIs:** * OpenAI (GPT-4o) for curated descriptions
    * Geocoding API for location data
    * Weather API for atmospheric shifts
